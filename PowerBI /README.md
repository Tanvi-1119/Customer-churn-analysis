# Power BI Dashboard

## Dashboard Overview

This Power BI dashboard was created to analyze customer churn patterns in the IBM Telco Customer Churn dataset. The dashboard provides an interactive view of customer behavior, churn trends, and high-risk customer segments.

## Dashboard Structure

### Page 1 – Executive Overview

This page provides a high-level summary of the business performance and customer churn situation.

**Visuals included:**

* Total Customers 
* Total Churned Customers 
* Churn Rate % 
* Average Monthly Charges 
* Churn Distribution 
* Customers by Contract Type
* Churn by Contract Type

### Page 2 – Churn Analysis

This page focuses on identifying the main factors contributing to customer churn.

**Visuals included:**

* Churn by Internet Service
* Churn by Payment Method
* Churn by Tenure
* Monthly Charges vs Churn
* Filters for Contract Type, Gender, and Internet Service

### Page 3 – Customer Profile

This page analyzes customer demographics and their relationship with churn.

**Visuals included:**

* Gender Distribution
* Senior Citizen vs Churn
* Partner vs Churn
* Dependents vs Churn
* Paperless Billing vs Churn

## Key Insights

* The overall customer churn rate is **26%**.
* Month-to-month contract customers contribute **23.5%** of the total customer base churn.
* Fiber optic service users contribute **18.42%** of the total customer base churn.
* Senior citizen customers contribute **6.76%** of the total customer base churn.
* Service dissatisfaction and inadequate product guidance appear to be significant contributors to customer churn.
